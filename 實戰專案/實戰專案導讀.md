## 💡 學習技巧

### 技巧 1：對比學習

```
登入範例（完整） ←→ 購物車（練習）
    ↓                    ↓
  研究結構           模仿實作
  理解原理           親手練習
```

**方法：**
1. 看登入的規格 → 看購物車的規格
2. 看登入的測試 → 寫購物車的測試
3. 看登入的實作 → 寫購物車的實作

### 技巧 2：小步前進

不要一次寫完所有功能：

```
❌ 錯誤做法：
寫完所有測試 → 寫完所有實作 → 一起測試
（太困難，容易卡關）

✅ 正確做法：
寫一個測試 → 實作讓它通過 → 重複
（小步前進，持續成就感）
```

**階段 1：理解規格**

## 🆘 遇到問題怎麼辦？

### 問題 1：不知道如何開始測試

**解決：**
1. 參考登入範例的測試結構
2. 從最簡單的測試開始
3. 使用 AI 生成測試骨架

### 問題 2：測試一直失敗

**解決：**
1. 查看 [錯誤訊息速查](../學習支援/錯誤訊息速查.md)
2. 使用 console.log 除錯
3. 簡化測試到最小可運作版本

### 問題 3：實作不知道怎麼寫

**解決：**
1. 看登入範例的實作方式
2. 先寫能通過測試的最簡單版本
3. 再逐步優化

### 問題 4：覺得太難想放棄

**解決：**
1. 暫停一下，深呼吸
2. 降低目標（先完成一個場景就好）
3. 查看 [學習卡關怎麼辦](../學習支援/學習卡關怎麼辦.md)
4. 在社群求助
