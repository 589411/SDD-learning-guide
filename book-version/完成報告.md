## 🚀 使用方式

### 第一次使用（5 分鐘）

```bash
# 1. 進入腳本目錄
cd book-version/scripts

# 2. 安裝依賴
npm install

# 3. 驗證配置
npm run validate

# 4. 生成書籍
npm run generate:leanpub

# 5. 查看結果
open ../manuscript/
```

### 日常更新流程

```bash
# 1. 更新 Repo 內容（在主目錄）
vim ../模組內容/模組0_規格驅動思維.md

# 2. 重新生成書籍
cd book-version/scripts
npm run generate:leanpub

# 3. 檢查輸出
cat ../manuscript/chapter-01.md
```

### 發布到 Leanpub

```bash
# 1. 生成書籍
npm run generate:leanpub

# 2. 上傳 manuscript/ 目錄到 Leanpub
# （使用 Git 或 Dropbox 同步）
```
